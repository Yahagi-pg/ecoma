= えこま

== えこまとは
えこまはえこまーけっとの略称であり、オンラインゲーム「エミルクロニクルオンライン」の取引掲示板です  
ゲーム内の取引補助機能が貧弱なため、そうしたサイトが欲しいと思ったため自分で作ることにしました  

== 主な機能
*記事の検索  
*記事の詳細表示  
*記事の新規投稿  
*記事への返信  
*記事の終了  


== 検討中の機能
*関心のある記事をブックマークする機能（チェックリスト機能）  
*自分の立てた記事、返信した記事に返信や終了などの変化があった際に通知する機能  
（通知対象はサイト内アカウント、メールアドレス等から検討中）
